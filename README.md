## 📈 Ruta de Aprendizaje
- **Git**
	- git --version: Se usa para verificar que Git está bien instalado en tu Windows.
	- git config: Se usa para identificarte. Es tu "firma" (nombre y correo) para que GitHub sepa a quién darle los puntos verdes.
	- git init: Se usa para "encender" Git en una carpeta. Convierte una carpeta normal en un Repositorio.
	- git status: Se usa para ver qué está pasando. Te dice qué archivos cambiaste y qué te falta por guardar.
	- git add .: Se usa para preparar los archivos. Es como poner tus documentos dentro de un sobre antes de sellarlo.
	- git commit -m "mensaje": Se usa para guardar permanentemente. Es el sello del sobre con una nota que explica qué hay dentro.
	- git remote add origin: Se usa para "tender un puente". Conecta la carpeta de tu computadora con tu repositorio en la web de GitHub.
	- git push: Se usa para enviar. Sube tus cambios de la PC a la nube de GitHub (aquí es donde aparece el color verde).
	- git pull: Se usa para bajar. Trae los cambios que hiciste en la web hacia tu computadora.
	- git clone: Se usa para descargar. Copia un repositorio completo de internet a una carpeta nueva en tu PC.
